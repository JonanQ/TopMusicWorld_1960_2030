# Top 30 UK (GitHub Pages)...

Este repositorio contiene el HTML de la lista Top 30 Reino Unido con reproductor integrado y fallback automático.

## 📌 Publicar en GitHub Pages
1. Ve a **Settings → Pages**.
2. En **Source**, selecciona rama `main` y carpeta `/ (root)`.
3. Guarda los cambios y tu web estará disponible en minutos.

## 🚀 Actualización automática con GitHub Actions
Este repositorio incluye un workflow que publica automáticamente en Pages cada vez que hagas `git push`.
